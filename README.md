# Python Job Bot

## Objective

Learn Python by building a job application bot.

This project is made to practice:

- Python fundamentals
- CSV file handling
- Email generation
- Automation concepts

## Features

- CSV file reading
- Duplicate email prevention
- Email generation
- Automatic email sending
- Web scraping

## Project Structure

```txt
companies.csv
main.py
sent.csv
README.md
```

## Workflow

```txt
CSV
↓
Read companies
↓
Generate emails
↓
Check duplicates
↓
Send emails
↓
Save sent emails
```

## Technologies

- Python
- CSV

## Run

```bash
python main.py
```
